Resumen

Este proyecto tiene como objetivo desarrollar un sistema basado en ESP32 para la lectura de tarjetas RFID y el almacenamiento de datos en una base de datos, con una interfaz web para la visualización de los registros.

Objetivo General

Diseñar e implementar un sistema de control de asistencia basado en tecnología RFID y ESP32, con almacenamiento y visualización en una plataforma web.

Objetivos Específicos

  • Desarrollar un prototipo funcional con tarjetas RFID.
  • Programar el ESP32 para comunicarse con una base de datos remota.
  • Diseñar una base de datos estructurada.
  • Crear una interfaz para visualizar los registros.
  • Optimizar la transmisión de datos.
  • Aplicar medidas básicas de seguridad.

Cronograma de Actividades

  • Semanas 1-2: Investigación sobre RFID y ESP32.
  • Semanas 3-4: Configuración inicial y pruebas.
  • Semanas 5-6: Desarrollo de código y conexión con base de datos.
  • Semanas 7-10: Desarrollo y prueba de interfaz web.
  • Semanas 11-12: Integración y pruebas finales.
  • Semanas 13-14: Documentación y presentación.
